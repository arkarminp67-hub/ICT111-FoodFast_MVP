# User Consent Statement — FoodFast

By using the FoodFast pre-ordering application, you agree to the collection of your Student ID, Name, and Mobile Phone Number solely for the purpose of order processing and pickup verification at campus canteen stalls. 

* Your data will not be shared with external third-party advertisers.
* Order history data used for canteen analytics is strictly anonymized.
* You may opt-out or request data deletion by contacting the system administrator.
